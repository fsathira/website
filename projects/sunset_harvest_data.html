Sunset Harvest Data

Interactive fermentation dashboard for Sunset Cellars (Suisun Valley, Fairfield, CA). Tracks Brix, must temperature, and Fairfield weather by fermentation day — live from Google Sheets.

Open Dashboard GitHub
Live app: The dashboard runs on the Stanford CS server at http://<machine>.stanford.edu:8050. If it doesn't load, the server process may be down — see the README for how to restart it, or connect via Stanford VPN if you're off-campus.

What it shows

  • Brix (sugar) vs. fermentation day
  • Must temperature vs. fermentation day
  • Fairfield air temp & rain (Open-Meteo) over the same window
  • Per-point notes on hover and click

Data sources

  • Live from Google Sheets (2025-first layout)
  • Falls back to local CSV if credentials are absent
  • Weather via Open-Meteo — no API key needed
  • Vintage → varietal → bin selection in the UI

Filters

Vintage, Varietal / Vineyard, and Bin dropdowns filter to a single fermentation lot. Day 0 = first recorded measurement for that lot.

Stack

  • Python 3.10+, Dash 2.14+, Plotly 5.18+
  • gspread + google-auth for live Sheets
  • Cormorant Garamond / Lato typography
  • Sunset Cellars palette (burgundy, cream)

Run it locally

# clone
git clone https://github.com/fsathira/SunsetHarvestData.git
cd SunsetHarvestData

# one-time setup
./setup.sh

# run (uses sample CSV if no Google credentials)
./run.sh --no-sheets
# open http://127.0.0.1:8050

Run on Stanford CS server

# SSH in — note the exact machine name
ssh <sunet>@corn.stanford.edu
hostname                        # e.g. corn30.stanford.edu

# clone into your web directory and set up
cd ~/WWW
git clone https://github.com/fsathira/SunsetHarvestData.git projects/sunset_harvest_data
cd projects/sunset_harvest_data
./setup.sh

# run inside screen so it survives logout
screen -S sunset-harvest
./run.sh --host 0.0.0.0 --no-debug
# Ctrl-A D to detach   |   screen -r sunset-harvest to reattach
