# Emoticon History

## Theme Toggle Icons

### Original Theme (Archived)

| Context   | Emoticon | Unicode | HTML Entity | Description              |
|-----------|----------|---------|-------------|--------------------------|
| Dark mode | ☾        | U+263E  | `&#9790;`   | Last Quarter Moon        |
| Light mode| ☀        | U+2600  | `&#9728;`   | Black Sun with Rays      |

These were the default theme toggle icons prior to the AI/neural theme update.
Used in: `js/main.js`, `index.html`, `playground.html`, `publications.html`,
`updates.html`, `projects/commute_optimizer.html`.

### Current Theme (AI/Neural — active as of 2026-03-05)

| Context   | Emoticon | Unicode | Description                                       |
|-----------|----------|---------|---------------------------------------------------|
| Dark mode | ⬡        | U+2B21  | White Hexagon — neural network node / molecular   |
| Light mode| ⬢        | U+2B22  | Black Hexagon — inverted, signals light mode      |

Hexagons were chosen to reflect the site's AI and computational biology theme:
